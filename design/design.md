# Processor
---

8-Bit processor
16-Bit memory addresses, 8-Bit entries

## Registers
### General Purpose
A - 8-bit
B - 8-bit
C - 8-bit

### System
PC - 16-bit Program counter
INS - 8-bit Instruction register
MAR - 16-bit Memory address register
AAR - 16-bit Argument address register

### Flags - 8-bit
ZF - Zero flag
CF - Carry flag
OF - Overflow flag
EF - Equal flag
x
x
IF - Interrupt flag
HF - Halt flag
